# 📊 Oracle SQL Tuning Advisor with Gemini API

Oracle 데이터베이스의 `V$SQL` 부하 쿼리를 실시간으로 모니터링하고, 통계 및 실행 계획을 수집하여 **Gemini AI(gemini-2.5-flash)**를 통해 구체적인 튜닝 가이드를 제공하는 데스크톱 GUI 애플리케이션입니다.

본 도구는 기업 내 업무용 테이블(기본값: `TB_` 프리픽스)을 주로 다루는 DBA 및 백엔드 개발자가 복잡한 SQL 성능 문제를 빠르게 분석하고 개선안(인덱스 설계, SQL 재작성 등)을 도출할 수 있도록 돕습니다.

---

## ✨ 주요 기능

1. **① SQL 분석 (V$SQL 모니터링)**
   * 경과시간, 평균시간, 버퍼 I/O, 디스크 I/O, 실행횟수 기준 **TOP 30 부하 쿼리**를 실시간으로 수집 및 표(Treeview) 형태로 정렬합니다.
   * 업무용 테이블(`TB_`) 위주의 쿼리 필터링 기능을 내장하여 시스템 노이즈 쿼리를 차단합니다.
   * 특정 `SQL_ID`를 더블클릭하여 대용량 CLOB 타입의 SQL 전문을 원클릭으로 조회할 수 있습니다.

2. **② SQL 입력 및 테이블 파싱**
   * 분석할 SQL을 자유롭게 입력 및 수정할 수 있으며, 주석 제거 후 `FROM/JOIN` 구문에서 대상 테이블을 정확히 파싱합니다.

3. **③ 인덱스 및 통계 정보 일괄 조회**
   * 파싱된 테이블의 인덱스 구조(`DBA_INDEXES`, `DBA_IND_COLUMNS`)와 테이블 통계(`NUM_ROWS`, `BLOCKS` 등), 인덱스 통계(`BLEVEL`, `DISTINCT_KEYS`, `CLUSTERING_FACTOR`)를 버튼 하나로 일괄 수집합니다.

4. **④ 실행계획(Explain Plan) 실시간 추적**
   * 입력된 SQL에 고유한 추적 태그(`TUNING_TAG`) 주석을 자동 삽입하여 실행합니다.
   * 실행 즉시 `V$SQL`에서 해당 쿼리의 실제 자원 사용량과 `DBMS_XPLAN.DISPLAY_CURSOR`를 통한 정확한 런타임 실행계획을 역추적합니다.

5. **⑤ 프롬프트 자동 최적화 및 Gemini API 분석**
   * 수집된 원본 SQL, 인덱스 상태, 테이블/인덱스 통계, 실행계획을 종합하여 최적의 튜닝 프롬프트를 조립합니다.
   * `gemini-2.5-flash` 모델을 통해 **문제점 분석 / 인덱스 개선안 / 통계 이슈 / SQL 재작성안 / 예상 효과**를 포함한 고품질의 한국어 가이드를 출력합니다.

6. **🎨 미려한 개발자용 UI/UX**
   * Catppuccin 스타일의 다크 테마가 적용되어 장시간 분석 시에도 눈의 피로를 최소화합니다.
   * 마크다운 렌더러가 내장되어 있어 AI가 답변한 마크다운 문법(코드 블록, 표, 강조 등)을 GUI 상에서 가독성 있게 표현합니다.

## 🚀 시작하기

### 1. 필수 요구사항
본 프로그램은 Oracle 데이터베이스와의 연결을 위해 JDBC 드라이버(`ojdbc8.jar`)를 사용하며, AI 분석을 위해 Gemini API Key가 필요합니다.

### 2. 필수 패키지 설치
프로그램 실행에 필요한 파이썬 패키지를 설치합니다.
```bash
pip install jaydebeapi jpype1 google-genai sqlparse
